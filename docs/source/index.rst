
pyTRSplat
=========

.. automodule:: pytrsplat

An extension for the
`pyTRS library <https://github.com/JamesPImes/pyTRS//>`_
for generating plats of
.

Feel free to reach out via
`my GitHub <https://github.com/JamesPImes/pyTRSplat//>`_
with feedback or inquiries, or to let me know if you run into any
issues.

You may want to reference the
`quickstart guide for the pyTRS library <https://github.com/JamesPImes/pyTRS/blob/master/guides/guides/quickstart.md>`_
to see how the parsing works, and in particular the
config parameters.
